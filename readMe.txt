A simple but effecive application design to create Canvass in a collaborative software environment.

To get started -- create a new profile.

Choose from one of five UX canvass.

* Business Canvass
* User Journey Canvass
* User Persona Generator
* Brand Board Blueprints
* Workflow Tables 


*** All canvas models are sourced from CC sourced canvass. 
**** Built on the UIKIT package. 


MIT Licensed.
